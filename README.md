# 🎮 لعبة طابق الأزواج - Memory Match Game

لعبة ذاكرة ممتعة واحترافية مبنية باستخدام **React** و **Vite** و **PeerJS**. تتميز بتصميم عصري (Dark Mode) وتدعم اللعب الجماعي أونلاين.

## ✨ المميزات الرئيسية

*   **🌍 تعريب كامل**: واجهة مستخدم عربية بالكامل مع دعم اتجاه اليمين لليسار (RTL).
*   **👥 اللعب الجماعي (Multiplayer)**: العب مع أصدقائك في نفس الوقت باستخدام كود غرفة بسيط، دون الحاجة لإنشاء حساب.
*   **👤 نظام الهوية**: اختر اسمك وسجل دخولك ليرى أصدقاؤك وجودك وسكورك لحظياً.
*   **🎨 تصميم بريميوم**: شاشة دخول (Lobby) مميزة مع تأثيرات بصرية جذابة وحركات انتقالية سلسة.
*   **📱 متجاوبة**: تعمل بكفاءة على جميع أحجام الشاشات.

## 🛠️ التقنيات المستخدمة

*   **React**: لبناء الواجهة التفاعلية.
*   **TypeScript**: لضمان جودة الكود واستقراره.
*   **Vanilla CSS**: لتصميم عصري ومرن (Glassmorphism & animations).
*   **PeerJS**: للمزامنة واللعب الجماعي (P2P).

## 🚀 التشغيل المحلي

1.  قم بتحميل المستودع:
    ```bash
    git clone https://github.com/ChouaibAlkemi/game-joke.git
    ```
2.  قم بتثبيت المكتبات:
    ```bash
    npm install
    ```
3.  شغل اللعبة:
    ```bash
    npm run dev
    ```
